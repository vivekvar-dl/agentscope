✨ Features

⚡

One-Line Integration

Wrap your agent with AgentScope and start monitoring instantly. Zero configuration required.

💰

Cost Tracking

Track LLM API costs in real-time. Supports GPT-4, Claude, Gemini, and 15+ models.

🔧

Tool Monitoring

See which tools your agent calls, how often, and their success rates.

⏱️

Performance Metrics

Track execution time, token usage, and decision counts for every run.

🎬

Replay & Export

Export execution traces to JSON and replay them for debugging.

🎨

Beautiful CLI

Color-coded output with Rich library. Pretty tables and live updates.

💻 Simple to Use

from agentscope import AgentScope from langchain.agents import AgentExecutor # Your normal agent setup agent = AgentExecutor(...) # Wrap with AgentScope (one line!) with AgentScope(agent, verbose=True) as scope: result = agent.run("Analyze this data") # Get instant insights print(f"Cost: ${scope.cost:.4f}") print(f"Tokens: {scope.tokens}") print(f"Tools: {scope.tool_calls}") # Export for later analysis scope.export("execution_trace.json")
