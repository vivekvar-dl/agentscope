## [0.1.0] - 2026-03-17

### Added
- 🎉 Initial release of AgentScope
- Core `AgentScope` context manager for agent monitoring
- Real-time metrics tracking (tokens, cost, duration, tool calls)
- Cost calculator for 15+ LLM models (GPT-4, Claude, Gemini, etc.)
- LangChain adapter with automatic agent wrapping
- Beautiful CLI output using Rich library
- JSON export and import for execution traces
- Compare functionality for side-by-side agent analysis
- Comprehensive test suite with pytest
- Professional documentation (README, CONTRIBUTING, INSTALLATION)
- CI/CD pipeline with GitHub Actions
- Pre-commit hooks (Black, Ruff, MyPy)
- GitHub Pages website
- MIT License

### Framework Support
